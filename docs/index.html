🔬 1. 分析手法

町丁のポリゴン（境界）データを使用し、各町丁内に人口が一様に分布していると仮定して分析しました。

一様分布モデルの概要

  • グリッド生成: 各町丁ポリゴン内に50m間隔でグリッド点を生成
  • カバー判定: 各グリッド点が最寄りAEDから300m以内かを判定
  • カバー率算出: カバーされたグリッド点の割合（0〜100%の連続値）
  • リスク調整: 年齢別心停止リスクで重み付けし、総人口と一致するよう正規化
カバー率 = カバーされたグリッド点数 ÷ 全グリッド点数
リスク調整人口 = リスク加重人口 × 正規化係数（= 総人口 / リスク加重人口合計）
📊 300mカバー範囲: 心停止から除細動までの理想的な時間（3分以内）を考慮し、カバー範囲を300mに設定。この基準で総人口カバー率96.2%、リスク調整後95.7%と算出されました。川崎市は高いAEDカバー率を達成していますが、残り約5.8万人のカバー外人口への対応が課題です。

⚠️ 2. 年齢別リスク加重

心停止発生率は年齢によって大きく異なります。高齢者ほど発生率が高いため、年齢構成を考慮したリスク加重を適用しました。

📊 リスク加重の意義: 単純な人口分布ではなく、「心停止が実際に発生しやすい場所」を特定することで、AED設置の効果を最大化します。

リスク重みの計算方法

リスク重み = 各年齢の発生率 ÷ 40〜44歳の発生率（45/10万人）

例: 75〜79歳 → 450 ÷ 45 = 10.0
（75歳以上は40代と比べて約10倍のリスク）
📈 リスク加重人口の正規化: リスク加重後の人口合計が総人口と一致するよう正規化係数（約0.15）を乗じています。これにより「約11,500人を新たにカバー」といった数値が実際の人口スケールで解釈できます。

🔍 現地調査による検証

OpenStreetMapデータを用いて各推奨地点の現地状況を調査し、以下を確認しました：

  • 全10地点で住宅地を確認: 各地点300m圏内に300〜1,000棟の建物が存在
  • コンビニ/スーパーあり: 4地点（2,4,7,8位）→ 店舗への設置が容易
  • 公共施設・屋外型: 6地点（1,3,5,6,9,10位）→ 町内会館・電柱型AED等への設置を推奨
  • 農地/森林混在: 一部地点で確認（住宅と混在、居住実態あり）

✅ 結論: 全地点で居住実態を確認。提案の信頼性は高い。
