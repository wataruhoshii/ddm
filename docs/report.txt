<タイトル>
川崎市AED最適配置分析レポート ～将来人口推計に基づく科学的なAED配置最適化提案～
</タイトル>
<要旨（アブストラクト）>
川崎市におけるAED（自動体外式除細動器）の配置状況を分析し、将来人口推計（2025〜2070年）と年齢別心停止リスクを考慮した最適配置を提案した。各町丁内に人口が一様に分布していると仮定し、50m間隔のグリッド点ごとにAEDカバー状況（300m圏内）を判定する手法を採用。分析の結果、現状のカバー率は総人口ベースで96.2%、リスク調整後で95.7%であり、約5.8万人がカバー外であることが判明した。優先度の高い10地点へのAED設置により、約2.7万人を新たにカバーし、カバー率を97%超に向上させることが可能である。
</要旨（アブストラクト）>
<利用したデータ>
【AEDデータ（計3,513件・重複除去後）】
・川崎市オープンデータ: 759件（セブン-イレブン208件含む）
・全国AEDマップ（日本救急医療財団）: 667件
・AED N@VI / aedm.jp: 2,433件

【人口・地理データ】
・町丁ポリゴン（676件）: e-Stat 小地域境界データ（令和2年国勢調査）
・将来人口推計: 川崎市（R7〜R52、5年ごと10時点）
・心停止発生率（年齢別）: 東京消防庁「令和5年 救急活動の現況」図表3-4, 3-5
</利用したデータ>

<目的・背景>
心停止は発生から除細動までの時間が生存率に大きく影響し、1分遅れるごとに生存率が約10%低下する。
理想的には3分以内（徒歩約300m圏内）にAEDへアクセスできることが望ましい。
川崎市は人口約152万人の政令指定都市であり、高齢化の進行に伴い心停止リスクの増加が見込まれる。
本分析は、現状のAED配置の評価と、将来の人口構成変化を見据えた最適配置提案を目的とする。
</目的・背景>
<仮説>
1. 川崎市内には300m圏内にAEDがない「空白地帯」が存在する
2. 高齢者が多く居住する地域では、人口あたりの心停止リスクが高い
3. 年齢別リスク加重を適用することで、単純な人口分布よりも効果的な設置優先順位が導出できる
4. 将来人口推計を考慮することで、長期的に有効な設置計画が立案できる
</仮説>
<現状分析>
【カバー状況】
・総人口: 約152万人（将来人口推計10時点（2025年〜2070年）の平均）
・総人口ベースカバー率: 96.2%
・リスク調整後カバー率: 95.7%
・カバー外人口: 約5.8万人
・AED総数: 3,513件
・全グリッド点: 70,726点（50m間隔）
・未カバーグリッド点: 22,172点（31.3%）

【分析手法】
・各町丁ポリゴン内に50m間隔でグリッド点を生成
・各グリッド点が最寄りAEDから300m以内かを判定
・カバー率 = カバーされたグリッド点数 ÷ 全グリッド点数
・年齢別心停止リスクで重み付けし、総人口と一致するよう正規化

【年齢別リスク重み（抜粋）】
・40〜44歳: 1.00（基準）
・75〜79歳: 11.63
・95歳以上: 72.24
※高齢者ほど心停止発生率が高く、75歳以上は40代の約10倍以上のリスク
</現状分析>
<課題の特定>
1. 約5.8万人（カバー外人口）が300m圏内にAEDがない状態
2. 高津区・麻生区・中原区・宮前区・多摩区に未カバーエリアが集中
3. 住宅地では商業施設が少なく、AED設置場所の確保が困難な地点が存在
4. 2025年→2070年で高齢化率が20.7%→31.3%に上昇する見込みであり、心停止リスクの増加が予測される
</課題の特定>

【現地調査結果】
・全10地点で住民の居住を確認（各地点300m圏内に300〜1,000棟の建物が存在）
・5地点はコンビニ/スーパーへの設置が容易
・残り5地点は住宅街（商業施設少）のため、屋外設置型AEDの導入が期待される
</分析結果>
<提案・改善策>
【短期施策（優先度高）】
1. 高津区久末への早急なAED設置（約7,200人をカバー、最大効果）
   - 設置候補: 町内会館・自治会館、または屋外設置型AED
2. 麻生区片平四丁目への設置（約3,200人をカバー）
   - 設置候補: たちばな幼稚園

【中期施策】
3. 残りTOP10地点への順次設置
   - コンビニ/スーパーがある地点: 店舗との協力によるAED設置
   - 商業施設がない地点: 町内会館・自治会館・屋外設置型AEDの検討

【長期施策】
4. 将来人口推計に基づく定期的な再評価（5年ごと）
5. 高齢化が進む地域への重点的なAED整備

【期待効果】
・TOP10設置により約2.7万人を新たにカバー
・カバー率: 95.7% → 97%超に向上
</提案・改善策>
