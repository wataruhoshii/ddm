# 03_analysis - AED最適配置分析

AEDデータと人口データを組み合わせて、最適配置を分析するフォルダです。

## 📁 ファイル構成

### スクリプト

| ファイル | 説明 |
|----------|------|
| `aed_optimization.py` | AED最適配置分析スクリプト |

### 出力ファイル

| ファイル | 説明 |
|----------|------|
| `aed_optimization_result.csv` | 分析結果（CSV） |
| `aed_optimization_analysis.png` | 分析グラフ |

## 📊 分析内容

### 算出する指標

| 指標 | 説明 |
|------|------|
| 人口/AED | 総人口をAED数で割った値 |
| 高齢者/AED | 65歳以上人口をAED数で割った値 |
| リスクスコア | AED不足度を0-100で表現 |
| 推奨追加AED | 目標達成に必要な追加数 |

### リスクスコアの計算式

```
リスクスコア = 高齢化率_norm × 0.3 
            + 人口AED比_norm × 0.3 
            + 高齢者AED比_norm × 0.4
```
