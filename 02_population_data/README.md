# 02_population_data - 人口データ取得

川崎市の人口データを取得するフォルダです。

## 📁 ファイル構成

### スクリプト

| ファイル | 説明 |
|----------|------|
| `fetch_population.py` | e-Stat APIから区別人口データを取得 |
| `fetch_chocho_population.py` | **川崎市から町丁別年齢別人口データを取得** |

### データファイル

| ファイル | 説明 |
|----------|------|
| `kawasaki_chocho_age_processed.csv` | **📌 町丁別年齢別人口（整形済み・分析用）** |
| `kawasaki_chocho_age_utf8.csv` | 町丁別年齢別人口（元データ・UTF-8） |
| `kawasaki_chocho_age.csv` | 町丁別年齢別人口（元データ・Shift-JIS） |
| `kawasaki_population_age3.csv` | 区別年齢3区分人口（e-Statから取得） |
| `kawasaki_population.xlsx` | 推計人口（参考） |

## 🛠️ 使用方法

### 町丁別データ取得

```bash
# データダウンロード・整形
python fetch_chocho_population.py
```

### 区別データ取得（e-Stat API）

```bash
# APIキーを設定してスクリプト実行
python fetch_population.py
```

※ e-Stat APIを使用する場合は、スクリプト内の `API_KEY` を自身のキーに置き換えてください。

## 📄 データソース

- **町丁別年齢別人口**: https://www.city.kawasaki.jp/170/page/0000181218.html
- **e-Stat（政府統計）**: https://www.e-stat.go.jp/
